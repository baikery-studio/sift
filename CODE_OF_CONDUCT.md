# Code of Conduct

## Our standard

This project welcomes anyone who wants to make it better. Be direct, be kind, and assume good
faith. Critique the code, not the person. We care about evidence over ego.

Unacceptable: harassment, personal attacks, demeaning comments, or sustained disruption.

## Scope

Applies in all project spaces, issues, pull requests, discussions, and any channel where you
represent the project.

## Enforcement

Report concerns privately to a maintainer via a private report on this repository (GitHub "Report a vulnerability" / Security Advisories), or by flagging a maintainer directly. Reports are handled confidentially.
Maintainers may remove comments, reject contributions, or block accounts that violate this
standard, at their discretion.

This is a lightweight adaptation in the spirit of the [Contributor Covenant](https://www.contributor-covenant.org).
